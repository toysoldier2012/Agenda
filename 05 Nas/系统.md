
## 配置

**网络配置**
DNS 信息：/etc/resolv.conf

**更新**
更新源列表 /etc/apt/sources.list

## 命令

版本号

```
lsb_release -a
```

本机信息

```
uname -a
```

ip 信息

```
ip a
```

软件安装

```
sudo apt update
sudo apt upgrade
sudo apt list --installed | grep program_name
```

Docker

```shell
sudo docker exec -it xxx bash

//查看docker网络
sudo docker network ls
sudo docker network inspect bridge | less

```
